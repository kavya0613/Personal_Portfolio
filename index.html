My Project

Music Recommendation System by Facial Emotion

The emotion recognition model in this project is trained on the FER 2013 dataset, enabling it to detect seven different emotions. The project operates by capturing live video feed from a webcam, passing it through the trained model to obtain an emotion prediction. Based on the predicted emotion, the application retrieves a playlist of songs from Spotify using the Spotipy wrapper and recommends the songs by displaying them on the screen
